Rentman → Ronnie Materiaallijst Converter

Zet een Rentman Excel export om in het bestand dat Rentman Ronnie (Custom GPT) nodig heeft.

Gemaakt door Narrative voor Brownys Event Productions

1 Exporteer uit Rentman

Zorg dat deze zes velden aangevinkt zijn in je Rentman export template:

  • Naam (in database)
  • Verhuur/verkoop
  • Code
  • Locatie (zie tip)
  • Huidig aantal
  • ID

💡 Locatie staat niet standaard aan. In de kolomkiezer zoek je een veld dat begint met "Locatie" — bijvoorbeeld Locatie in standaard voorraadlocatie (onder VOORRAAD) of Locatie in magazijn per voorraadlocatie. De tool accepteert beide. Voeg het veld éénmalig toe aan je export template — daarna blijft 'ie aan.

2 Upload je Excel-bestand

Sleep je .xlsx bestand hier

of klik om te kiezen

3 Kopieer de output

4 Upload naar Rentman Ronnie

  1. Open Rentman Ronnie (je Custom GPT)
  2. Klik ⚙ (rechtsboven) → Edit GPT → Configure tab
  3. Scroll naar Knowledge
  4. Verwijder het oude brownys-materiaallijst-with-id.md
  5. Sleep het nieuwe bestand erin (of plak de gekopieerde tekst in een nieuw bestand)
  6. Klik Update rechtsboven
